Animation, State, Form exports are usable Vanilla Code.

Vanilla Animation export can be used anywhere. It is fully responsive. Webflow export includes Normalize HTML5 CSS reset, and a typography CDN, Webflow will also automatically source set/optimize images. Animations and micro interactions are collaborative with state/forms and API hooks in WYSWYG deployment versions. Static exports can also deploy to Web 3 enabled publishes.

Webflow / Code Sandbox are wonderful collaboration and preview design/code environments.

You do not need to "host" in Webflow, simply use as an add on to Figma/Principle (Figma Animations does not work with API/CMS and will not ship as responsive code) workflow. The enhanced code can be "shipped" to any export: Next/Vue-you can include VUE in Stack/Shopify/AEM/Saleforce.
